# Spotify Search
This is an app built from the Spotify API that allows the user to search for songs by title.

# Usage
Enter the song title into the search bar, then hit "search"

# Technologies Used
* HTML
* CSS
* Bootstrap
* JavaScript
* jQuery
* AJAX

# Existing features
* Search bar and search button seen
